GitのRepositoryをDropbox内に作成



概要

GitのRepositoryをDropbox内に作成する。

出来ると、まあ、ウマーって感じで。



手法

なんの事は無い、Dropboxの設定で、

マシン内にDropboxフォルダを作った後で、

通常のgitの所作を行うのみ。



操作

Dropbox内で、


mkdir origin.git

cd origin.git

git init --bare



コミットとか

他所のRepoで、

git remote add origin (Dropbox/とかマシン内のアドレス)origin.git

git commit -m "なにかしら" 変更を加えたファイル

git push origin master
